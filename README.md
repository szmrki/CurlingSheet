# CurlingSheet
## How to Use
### Clone this Repository
~~~cmd
git clone https://github.com/szmrki/CurlingSheet.git
cd CurlingSheet
~~~

### Create virtual environment
~~~cmd
python -m venv .venv
.venv\Scripts\activate.bat #Windows
(.venv) pip install -r requirements.txt
~~~

### Create exe file
~~~cmd
pyinstaller main.spec
~~~
